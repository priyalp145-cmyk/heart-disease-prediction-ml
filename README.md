# Heart Disease Prediction using Machine Learning

##  Project Overview
This project predicts the likelihood of heart disease using machine learning algorithms.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

##  Steps Performed
- Data Cleaning
- Handling Missing Values
- Label Encoding
- Feature Scaling
- Exploratory Data Analysis (EDA)

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

## Results
- Accuracy: ~81%
- Evaluated using Confusion Matrix & Classification Report
